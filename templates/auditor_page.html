How to get your HubSpot token

  1. Log into the HubSpot account you want to audit.
  2. Go to Settings.
  3. Open Integrations -> Private Apps, or Development -> Legacy apps if that is what your account shows.
  4. Create a new private app, or open an existing one.
  5. In the app’s auth/scopes section, enable the scopes listed below.
  6. Copy the private app access token. It will start with pat-.
  7. Paste the token into this page and run the audit.
