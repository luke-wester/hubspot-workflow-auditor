HubSpot Workflow Auditor

Paste a HubSpot private app access token and review workflow inventory, collisions, property touches, and workflow suggestions directly in the browser.

How to get your HubSpot token

  1. Log into the HubSpot account you want to audit.
  2. Go to Settings.
  3. Open Integrations -> Private Apps, or Development -> Legacy apps if that is what your account shows.
  4. Create one private app for this workflow auditor.
  5. Enable the workflow-related read/view scopes listed below.
  6. Copy the private app access token. It will start with pat-.
  7. Paste the token below and run the audit.

Required scopes

  • automation
  • crm.schemas.contacts.read
  • crm.schemas.companies.read
  • crm.schemas.deals.read
  • crm.schemas.tickets.read
  • crm.lists.read
